Gerador de Etiquetas – Sistema Web

Este é um sistema web para geração e impressão de etiquetas logísticas, permitindo inserir dados de pedidos e gerar etiquetas automaticamente para impressão.

O sistema cria múltiplas etiquetas com base na quantidade de volumes informada, facilitando o processo de expedição.

É um sistema prático e voltado para uso empresarial/logístico.

⚙️ Funcionalidades
✔ Cadastro de dados

O sistema permite inserir:

Número da Nota Fiscal

Número do Pedido

Nome do Cliente

Data

Quantidade de volumes

Peso

Transportadora

INDEX

✔ Geração automática de etiquetas

O sistema gera automaticamente:

Volume 1/10

Volume 2/10

Volume 3/10

Até completar o total informado.

Isso é feito via JavaScript com loop.

INDEX

Exemplo:

VOLUME: 3/10

Muito útil para logística.

Possui modo de impressão com:

Formato A4 paisagem

Margens definidas

Layout limpo
